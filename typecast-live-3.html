Microphone Blocked

Mic access was denied. Follow the steps below for your device, then click Try Again.

🖥️ Chrome / Edge on PC or Mac

1. Click the 🔒 lock icon in the address bar
2. Set Microphone → Allow
3. Press F5 to refresh, then click Try Again

📱 Chrome on Android

1. Tap 🔒 in Chrome's address bar
2. Tap Permissions → Microphone → Allow
3. Reload the page, then click Try Again

⚙️ Still blocked? Chrome Settings

Type chrome://settings/content/microphone in the address bar
Remove this file from the Blocked list → reload

After fixing the setting above, click Try Again — no need to re-download the file.
